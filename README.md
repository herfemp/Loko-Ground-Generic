# Loko-Ground-Generic
Base on micro pyhton code of the loko ground station to use with any esp32 but using c++ to use with LOKO tiny Gps Tracker
using same lora-e5 module from Seeed (https://www.seeedstudio.com/LoRa-E5-Wireless-Module-p-4745.html)

Base on : https://github.com/tomipiriyev/Loko/tree/main/Software/LokoGround%20Firmware
Can be configured with original app: https://github.com/tomipiriyev/Loko/tree/main/Software/LokoMobileApp
To track the original LoKo Air Transmitter decribed and  available at:https://www.nolilab.com/ and https://www.seeedstudio.com/Loko-Air-p-6447.html
